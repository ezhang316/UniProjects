# 2521: Data Structures and Algorithms (C)

Core data structures and algorithms course covering fundamental techniques for efficient computation. This course combines theory with practical C implementations of classic algorithms and data structures.

## Projects

### Graph Algorithms
**Dijkstra's Shortest Path & Hierarchical Clustering**

Implements Dijkstra's algorithm to compute shortest paths in weighted directed graphs. The solution uses a priority queue data structure to efficiently find minimum distances from a source vertex to all other vertices, returning predecessor lists for path reconstruction.

**Key Concepts:**
- Shortest path computation (Dijkstra's algorithm)
- Priority queue implementation
- Graph representation and traversal
- Dynamic memory management in C

### Trees & Information Retrieval 
**Inverted Index & Text Search**

Builds an inverted index data structure using binary search trees for efficient text retrieval. Implements TF-IDF (Term Frequency-Inverse Document Frequency) ranking to process multi-file document collections and retrieve ranked search results.

**Key Concepts:**
- Binary search trees (insertion, traversal, searching)
- Linked list structures for file lists
- Inverted indexing for information retrieval
- TF-IDF algorithm for relevance ranking
- File I/O and text processing

## Technical Highlights

- **Memory-safe C:** Proper dynamic allocation and deallocation across all data structures
- **Algorithm efficiency:** Focus on time and space complexity analysis
- **Graph algorithms:** Shortest path, graph traversal, and neighbor exploration
- **Search optimization:** TF-IDF ranking and efficient document retrieval
