# 3121: Algorithms and Programming Techniques

Advanced course in algorithm design and analysis, covering systematic approaches to solving computational problems. Focuses on proving correctness, analyzing complexity, and selecting appropriate algorithmic paradigms for different problem classes.

## Topics Covered

### Algorithm Analysis
Formal techniques for analyzing time and space complexity, asymptotic notation (Big-O, Omega, Theta), and worst/average/best-case analysis.

### Divide-and-Conquer
Recursive problem decomposition, merge sort, quicksort, and applications to searching and selection problems. Includes recurrence relation analysis and solving via master theorem.

### Greedy Algorithms
Greedy choice property and optimal substructure. Applications include activity selection, Huffman coding, fractional knapsack, and minimum spanning trees.

### Dynamic Programming
Optimal substructure identification, memoization vs. tabulation, classic problems including longest common subsequence, edit distance, matrix chain multiplication, and knapsack variants.

### Network Flow Algorithms
Maximum flow problems, min-cut max-flow theorem, Ford-Fulkerson method, and applications to bipartite matching and resource allocation.

### String Matching Algorithms
Pattern searching techniques including naive matching, KMP algorithm, Boyer-Moore, and Rabin-Karp hashing. Analysis of string algorithms and their applications.

### Linear Programming
Optimization problems with linear objectives and constraints. Simplex method, duality, and applications to resource allocation and scheduling.

### Intractable Problems and Approximation Algorithms
NP-completeness, NP-hard problems, and proving hardness. Approximation algorithms for hard problems with guaranteed solution quality bounds.

## Problem Set Structure

Assignments 1-4 progress through increasingly complex algorithm design challenges, requiring formal proofs of correctness and complexity analysis. Each assignment includes multiple problems testing different paradigms and requiring comparative analysis of solution approaches.

## Learning Outcomes

- Design efficient algorithms for diverse problem classes
- Prove algorithm correctness using induction and exchange arguments
- Analyze time and space complexity rigorously
- Select optimal algorithmic approaches for specific constraints
