# 3311: Database Systems (SQL and PL/pgSQL)

Relational database design, SQL querying, and stored procedure development using PostgreSQL. Covers database modeling, query optimization, and server-side programming with PL/pgSQL.

## Projects

### Assignment 1: Complex SQL Queries
**Brewery Database Analysis**

Complex multi-table SQL queries on a real-world brewery database. Demonstrates advanced SQL techniques for data retrieval and aggregation.

**Key Queries:**
- Filtering by location and temporal attributes (new breweries in Sydney in 2020)
- Multi-join pattern matching (beers with names matching their style)
- Aggregation and ranking queries (brewery history and popularity metrics)
- Window functions and subquery optimization

**Concepts:**
- Complex JOIN operations across multiple tables
- WHERE clause filtering and aggregate functions
- Subqueries and views for logical organization
- Query optimization and index awareness

### Assignment 2: Stored Procedures & Business Logic
**PL/pgSQL Development**

Server-side programming with stored procedures and functions. Implements database-level business logic, transaction handling, and procedural control flow.

**Key Concepts:**
- Procedural SQL (PL/pgSQL) programming
- Stored procedures and functions
- Transaction control and error handling
- Triggers and data integrity
- Performance optimization techniques

## Database Design Principles

- **Normalization:** Proper schema design avoiding redundancy
- **Foreign keys & referential integrity:** Maintaining data consistency
- **Query efficiency:** Index design and execution plan analysis
- **Transactions:** ACID properties and concurrency control
