# 9417: Machine Learning and Data Mining

Supervised learning and classification techniques using Python and Jupyter Notebooks. Covers the complete machine learning pipeline from data preprocessing through model evaluation and hyperparameter tuning.

## Group Project: Classification & Data Analysis

### Dataset & Problem
Multi-class classification problem on a real-world dataset. Project emphasizes proper data handling, feature engineering, and model selection.

### Techniques & Technologies

**Data Preprocessing:**
- Missing value imputation using scikit-learn's `SimpleImputer`
- Categorical feature encoding (one-hot encoding)
- Feature scaling and normalization (StandardScaler)
- Data exploration and visualization (pandas, seaborn, matplotlib)

**Feature Engineering:**
- Exploratory data analysis (EDA)
- Feature correlation analysis
- Feature selection and dimensionality reduction
- Train-test split with cross-validation

**Classification Models:**
- Logistic Regression for baseline performance
- Support Vector Machines (SVM) for non-linear classification
- Decision Trees and Random Forests for ensemble methods
- Model hyperparameter tuning using `RandomizedSearchCV`

**Evaluation Metrics:**
- Accuracy, precision, recall, F1-score
- Confusion matrices and classification reports
- Cross-validation scores
- Log loss for probabilistic predictions

### Key Libraries
- **scikit-learn:** Classification models, preprocessing, evaluation
- **pandas:** Data manipulation and analysis
- **NumPy:** Numerical computations
- **Matplotlib & Seaborn:** Data visualization
- **SciPy:** Statistical computations

### Workflow
1. Load and explore dataset
2. Handle missing values and encode features
3. Build preprocessing pipeline
4. Train multiple classification models
5. Evaluate and compare model performance
6. Optimize hyperparameters
7. Final model evaluation and reporting

### Learning Outcomes

- End-to-end machine learning workflow
- Proper train-test methodology and cross-validation
- Feature preprocessing and engineering
- Model selection and hyperparameter optimization
- Classification metrics and interpretation
