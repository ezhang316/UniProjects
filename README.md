# UNSW Computer Science Portfolio

A collection of university coursework projects from UNSW demonstrating core computer science fundamentals, data structures, algorithms, systems programming, databases, and full-stack development.

## Courses & Projects

### Foundational Programming
- **[1511: Programming Fundamentals (C)](./1511%20Programming%20Fundamentals%20(C)/)** — Introductory C programming with minesweeper implementation and file I/O-based pokedex system using linked lists.
- **[1521: Computer Systems Fundamentals (C, MIPS)](./1521%20Computer%20Systems%20Fundamentals%20(C,%20MIPS)/)** — Low-level systems programming in C and MIPS assembly, including minesweeper game and text encoding.
- **[1531: Software Engineering Fundamentals (Python)](./1531%20Software%20Engineering%20Fundamentals%20(Python)/)** — Collaborative Python development practices and software engineering principles.

### Object-Oriented & Systems Design
- **[2511: Object-Oriented Design & Programming (Java)](./2511%20Object-Oriented%20Design%20&%20Programming%20(Java)/)** — Java-based OOP design patterns and group software project.
- **[2521: Data Structures and Algorithms (C)](./2521%20Data%20Structures%20and%20Algorithms%20(C)/)** — Core DSA concepts including graph algorithms (Dijkstra's shortest path), binary search trees, and information retrieval systems.

### Intermediate & Advanced Topics
- **[3121: Algorithms and Programming Techniques](./3121%20Algorithms%20and%20Programming%20Techniques/)** — Advanced algorithmic problem-solving and algorithm analysis.
- **[3311: Database Systems (SQL and PL/pgSQL)](./3311%20Database%20Systems%20(SQL%20and%20PL%20pgSQL)/)** — Relational database design, complex SQL queries, views, and stored procedures.
- **[3331: Computer Networks and Applications (Java)](./3331%20Computer%20Networks%20and%20Applications%20(Java)/)** — Network programming and distributed systems concepts.
- **[3900: CS Capstone Project (TypeScript & React)](./3900%20CS%20Capstone%20Project%20(Typescript%20and%20React)/)** — Full-stack web application with React frontend, Spring Boot backend, and MySQL database.

### Advanced Topics
- **[6080: Web Front-end Programming (HTML, CSS, JavaScript, React)](./6080%20Web%20Front-end%20Programming%20(HTML,%20CSS,%20Javascript%20and%20React)/)** — Modern web development spanning vanilla JavaScript to React frameworks.
- **[6771: Advanced C++ Programming (C++)](./6771%20Advanced%20C++%20Programming%20(C++)/)** — Modern C++ techniques including templates, custom containers, iterators, polymorphism, and metaprogramming.
- **[9417: Machine Learning and Data Mining (Python & Jupyter)](./9417%20Machine%20Learning%20and%20Data%20Mining%20(Python%20and%20Jupyter%20Notebooks)/)** — Supervised learning, classification models, and data preprocessing with scikit-learn.

## Technical Skills Demonstrated

**Data Structures & Algorithms** — Binary Search Trees, graphs, linked lists, Dijkstra's algorithm, shortest path computation, inverted indexing

**Systems Programming** — Low-level C programming, MIPS assembly language, memory management, bit manipulation

**Object-Oriented Programming** — Java OOP design, design patterns, inheritance, polymorphism

**Databases** — SQL query optimization, complex joins, views, aggregate functions, stored procedures with PL/pgSQL

**Backend Development** — Spring Boot, RESTful APIs, database integration, Java enterprise patterns

**Frontend Development** — React, TypeScript, HTML/CSS, component architecture, state management

**Machine Learning** — Classification algorithms, feature preprocessing, cross-validation, model evaluation (scikit-learn, pandas)

**Full-Stack Development** — Multi-tier application architecture, API design, database interaction, authentication

---

Each course folder contains detailed documentation on specific projects, technical implementations, and key learnings. Projects are organized by coursework and include source code, assignments, and relevant course materials.
